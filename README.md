# EcoPedia
### Web Informasi Sampah • Pilah • Daur Ulang (Static Website)

EcoPedia adalah website edukasi publik yang membantu pengguna memahami cara memilah dan membuang sampah dengan benar.  
Project ini dibuat **tanpa database** (static web) menggunakan **HTML, CSS, dan JavaScript**, dengan UI bertema **Earth Accent (Hijau–Coklat)**.

</div>

---

## ✨ Highlights

- ✅ **Welcome Page (Full Page Hero)** untuk sambutan awal
- ✅ **Navbar Search + Auto Suggestion** (Google-like dropdown)
- ✅ **Dropdown Navigasi** yang clean dan rapi
- ✅ **Ensiklopedia Sampah** dengan **Search + Filter kategori**
- ✅ **Modal Detail** (penjelasan, cara buang, kesalahan umum, fakta singkat)
- ✅ **Smooth Scroll + Reveal Animation** untuk transisi yang halus
- ✅ **Back To Top Button** dengan desain bulat (hijau)
- ✅ **Responsive Layout** (nyaman dibuka di laptop & mobile)

---

## 🎯 Tujuan Project

EcoPedia dibuat untuk:
- menjadi media edukasi pemilahan sampah yang sederhana dan cepat dipahami
- menyediakan sistem pencarian sampah yang praktis (bukan sekadar artikel panjang)
- menjadi project portfolio yang menonjolkan **UI/UX + interaksi front-end**

---

## 🧩 Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**

> Tidak menggunakan framework & database agar ringan dan mudah dipelajari.

## 🚀 Cara Menjalankan
1) Jalankan langsung
- Buka file berikut di browser:

- index.html

2) (Rekomendasi) Menggunakan VS Code Live Server
- Install extension Live Server

- Klik kanan index.html

- Pilih Open with Live Server

## 🔎 Cara Pakai
- Gunakan search di navbar untuk mencari item sampah (contoh: baterai, styrofoam, minyak)

- Pilih hasil suggestion yang muncul

- Website akan mengarahkan ke bagian Ensiklopedia

- Klik item untuk melihat detail lengkap melalui modal

## 🗂️ Data Ensiklopedia
Data disimpan di file berikut:

- js/data.js

- Format data menggunakan array objek JavaScript agar:

- mudah ditambah/edit

- tetap ringan tanpa database
